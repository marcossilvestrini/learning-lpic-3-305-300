Deploy — Nginx em Docker

Checklist de time-to-live imediato:

# 1) Salve este arquivo como: index.html
# 2) Execute o container mapeando o volume estático do Nginx
docker run -d --name lpic305 -p 8080:80 \
  -v $(pwd)/index.html:/usr/share/nginx/html/index.html:ro \
  nginx:alpine

# Acesse: http://localhost:8080

Para hardening: utilize nginx:stable-alpine, desabilite autoindex, e sirva via TLS atrás de um reverse proxy corporativo.
